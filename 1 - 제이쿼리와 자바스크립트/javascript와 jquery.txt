 jquery : node/ 동적인 작업을 담당
     javascript를 쉽게 구현하기 위해 만들어놓은 플러그인
     javascript라이브러리중의 하나이다
     1.선택자
     2. 메소드 : 자바스크립트에 내장되어 있는 명령어, 사용자가 직접 제작하여 사용하는 명령어(함수)
     3. 이벤트 : 사용자 동작에 의해서만 나오는 작업(모션)
     4. node : 경로/가지치기

javascript : 동적인 작업을 담당하는 프로그램의 원본
         1. 변수 : 변하는 수
         2. 조건문
         3. 반복문
         4. 함수	

jquery를 사용하는 이유
1. 페이지 접근하는 방법이 쉽다.
2. 상호작용이 쉽다
3. 애니메이션 작업이 쉽다

** jquery를 사용하는 방법
1. 초기플러그인이 설정을 해야한다.
jquery.com접속하여 다운로드
2. 내부스크립트/외부스크립트
내부스크립트=>head태그안에 설정
<script src="./script/jquery-3.6.0.min.js"></script> 
첫번째 다양한 플러그인
사용자 스크립트

3. jquery 문법
- $(); jquery 접근하는 방법
- " " , ' ' : 문자를 묶어줄때 사용, 숫자는 묶지 않는다, javascript를 묶지 않는다.
- . : 연결자
- $("선택자").명령어();
-주석: // : 한줄주석
        /* */ : 여러줄주석
-마감처리는 소괄호 뒤에 사용한다.  ; - 마감처리
